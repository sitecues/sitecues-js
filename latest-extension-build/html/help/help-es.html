Sitecues Ampliación y Lectura: Ayuda

Sitecues hace más fácil ver, escuchar y usar este sitio web.

Amplíe toda la página web o las partes que más le interesan. Usted incluso puede escuchar a Sitecues leyendo el texto en voz alta.

Ampliación de Página

Con el ratón, mueva el cursor sobre el ícono de Sitecues y use el control deslizante para ampliar y alejar.

Usted simplemente también puede pulsar + o - para hacer ampliación con el teclado, por favor, vea la lista completa de los consejos de teclado más abajo.

Lectura One-Touch

La función de lectura One-Touch amplía su contenido seleccionado y también puede leérselo en voz alta. La lectura One-Touch está habilitada una vez que la ampliación o lectura de Sitecues está encendida.

Ampliar parte de una página web

  • Comience moviendo el ratón sobre algo que quiera leer. Cabe destacar (si no lo está activada, asegúrese de que la ampliación o lectura se haya activado en Sitecues).
  • Presione la barra espaciadora para ampliar el área resaltada.

Escuchar parte de una página web

En primer lugar, active la lectura:

  • Con el ratón, mueva el cursor sobre el ícono de Sitecues y haga clic en el botón de lectura
  • Resalte el texto que desea leer y a continuación, presione la barra espaciadora

Comandos de Teclado

Acercamiento de página

  • + para acercar
  • - para alejar
  • Pulse estas teclas durante más tiempo para acercar o alejar durante más tiempo

Acercamiento One-Touch

  • Si todavía nada está resaltado, presione la Barra Espaciadora para resaltar algo cerca del centro de la pantalla.
  • Mueva el resaltado con las teclas de flecha Arriba / Abajo / Izquierda / Derecha.
  • Presione la barra espaciadora para ampliar el área resaltada.
  • A continuación, pulse Escape para salir de la lectura One-Touch. Al pulsar Escape por segunda vez se borrará el resaltado.
  • Navegue rápidamente con H / Mayúsculas+H para moverse hacia adelante / atrás.

Lectura One-Touch (lectura)

  • Presione Alt+Apóstrofe para activar o desactivar la lectura.
  • Ahora, el comando One-Touch (barra espaciadora) también leerá el texto.

Explorando con Lectura

Presionar la tecla de Mayúsculas mientras mueve el resaltado le dice a Sitecues que lea las áreas resaltadas. Existen varias formas de usarlo:

  • Mayúsculas+teclas de flecha para mover y leer en la dirección dada
  • Mayúsculas mientras mueve el ratón para leer el texto sobre el que se mueva el ratón
  • Mayúsculas por sí mismo para leer el resaltado actual

Use mayúsculas para leer trabajos ya sea si la lectura está encendida o no en el panel de Sitecues.

Otros Atajos Geniales

Usted puede hacer una ampliación de la página con su ratón. Pruebe mantener pulsada la tecla Control mientras gira la rueda del ratón. Usted incluso puede pellizcar / soltar en un trackpad (Solo Chrome en Mac).

Saliendo de Sitecues

Para salir de Sitecues apague o bien el acercamiento y lectura o pulse Alt+0.
