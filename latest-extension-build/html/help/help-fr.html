Sitecues Zoom et Lecture: Aide

Sitecues rend plus facile à voir, entendre et utiliser ce site Web.

Zoom de la page Web entière ou les parties qui vous intéressent le plus. Vous pouvez même écouter Sitecues lire le texte à haute voix.

Zoom de page

Avec la souris, survolez l'icône Sitecues et utiliser le curseur pour zoomer et dézoomer.

Vous pouvez aussi simplement appuyer sur + ou - pour zoomer avec le clavier — s'il vous plaît voir la liste complète de conseils clavier ci-dessous.

Lecture One-Touch

La fonction Lecture One-Touch zoome sur votre contenu sélectionné — et peut également lire à haute voix pour vous. Lecture One-Touch est activée une fois Sitecues zoom ou la parole est activée.

Zoom d’une partie d'une page Web

  • Démarrez en déplaçant votre souris vers quelque chose que vous voulez lire. Il convient de souligner (si elle ne l’est pas, assurez-vous que le zoom ou la lecture ont été activé dans Sitecues).
  • Appuyez sur la barre d'espace pour agrandir la zone en surbrillance.

Ecoutez une partie d'une page Web

Tout d'abord, mettez-vous en mode lecture:

  • Avec la souris, survolez l'icône Sitecues et cliquez sur le bouton Voix
  • Sélectionnez le texte que vous voulez lire puis appuyez sur la barre d'espace

Commandes clavier Zoomer la page

Zoom de page

  • + pour agrandir
  • - pour rétrécir
  • Maintenez un appuie long sur ses touche pour plus de zoom et de dézoom

One-Touch Zoom

  • Si rien est encore en surbrillance, appuyez sur la barre d'espace pour mettre en évidence quelque chose près du centre de l'écran.
  • Placez le pointeur autour avec les touches fléchées Droite / Haut / Bas / Gauche.
  • Appuyez sur la barre d'espace nouveau pour agrandir la zone en surbrillance.
  • appuyez sur Echap pour fermer One-Touch Lecture. Appuyant sur Echap une deuxième fois pour annuler la surbrillance.
  • Naviguez rapidement avec H / Maj+H pour déplacer avant / arrière par rubrique.

One-Touch Lecture (Voix)

  • Appuyez sur Alt+Apostrophe pour activer ou désactiver la parole
  • Maintenant, la commande One-Touch (barre d'espace) pourra également lire le texte.

Explorer avec la voix

En appuyant sur la touche Maj tout en déplaçant le curseur indique à Sitecues de lire zones en surbrillance. Il y a plusieurs façons de l'utiliser:

  • Maj+flèche pour déplacer et lire dans la direction donnée
  • Maj tout en déplaçant la souris pour lire tout ce que le curseur survole
  • Maj seul pour lire la partie en surbrillance

UUtiliser shift pour la lecture peu importe si la commande vocale est activée dans le panneau Sitecues.

Autres raccourcis intéressants

Vous pouvez agrandir la page avec votre souris. Essayez maintenant la touche Ctrl enfoncée tout en roulant molette de votre souris. Vous pouvez même pincer / unpinch sur un trackpad (Chrome sur Mac uniquement).

Quitter Sitecues

Pour quitter Sitecues, soit désactiver les zooms ou la parole, ou appuyez sur Alt+0..
